# Tweet_Sentiment_Analysis
Sentiment Analysis

▪ Tweet Sentiment Analysis | Self Project | Supervised Machine Learning [May,2024 - July,2024]
▪ Classified 1.8 Lakh+ tweets with 13 different features into positive, negative, or neutral using Bag-of-Words and Tfidf Vectorizer
▪ Processed data with NLTK for comprehensive cleaning, including punctuation removal, lemmatization & stopword elimination
▪ Created detailed interactive visualizations using Plotly and WordCloud, enhancing insights from tweet sentiment analysis
